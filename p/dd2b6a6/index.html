CocoaPods版本升级

发表于 2016-07-05 更新于 2023-05-25 分类于 随记

今天和往常一样打开一个包含CocoaPods类库的项目

命令行执行

pod install --verbose --no-repo-update

[!] The 'master' repo requires CocoaPods 0.39.0 -

what，失败了~~😱，查看原因提示需要CocoaPods 0.39.0版本，于是乎查看了下机器上安装的CocoaPods版本。

命令行执行

pod --version

0.29.0

额~~ 看来的确是需要更新了😔

命令行更新步骤

sudo gem update --system  // 首先我们先更新下gem，在国内如果没有FQ的话 建议切换到淘宝源

gem sources --remove https://rubygems.org/  // 移除官方源切换到淘宝源
gem sources -a https://ruby.taobao.org/  

gem sources -l  // 查看源
\*\*\* CURRENT SOURCES \*\*\*
https://ruby.taobao.org/

sudo gem install cocoapods // 安装cocoapods

其实你应该发现到更新和安装过程是一样的，OK,我们现在再来看下CocoaPods版本~

命令行执行

pod --version

0.39.0

OK, 打完收工!!!

Now,Enjoy it 👻 👻 👻
